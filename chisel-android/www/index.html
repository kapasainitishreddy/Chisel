Facial Analysis

Look at yourself. Measure what matters.

A real 468-point face mesh runs on-device (Google MediaPipe). It reads symmetry, jaw, cheekbones, nose, lips and eyes, plus a dermatology-grade skin scan (Fitzpatrick tone via ITA°, CIELAB undertone, white-balanced off the sclera, glare-rejected median sampling, redness, T-zone shine, evenness, under-eye), a hair check and a teeth shade read — each paired with a science-backed remedy and its evidence strength, then tracked over time. Nose-hair and beard density aren't honestly readable from a selfie, so those ship as guided self-checks instead of a faked score.

Step 1

Plug in & set USB mode

Connect the phone to this PC. On the phone, pull down the notification shade and tap the USB notification. Choose PTP (Camera) or File transfer / MTP. Charging-only mode will not be detected by Android Studio.

Step 2

Verify USB debugging

Settings → Developer options → confirm USB debugging is on. The first time you plug in, the phone will pop a "Allow USB debugging" prompt — tap Allow. You can also check the box to always allow from this computer.

Step 3

Sideload the APK

From the chisel-android folder on this PC, run the included deploy.bat (Windows) or deploy.sh (Mac/Linux). It will build the APK and push it to your phone over USB. The icon appears in your launcher as Chisel.

Alternative

Just use it in the browser

If you'd rather skip the install, run npx serve in chisel-android/www and open the URL on your phone. Tap Chrome menu → Add to Home screen for a fullscreen, installable PWA.
